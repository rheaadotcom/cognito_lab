# CognitoLab

CognitoLab is a browser-based cognitive research platform that allows researchers to design, manage, and run behavioral experiments online. Participants can complete tasks through their browsers while the system collects responses and stores them securely for analysis.

The platform makes psychological and cognitive experiments easy to deploy, scalable, and accessible remotely.

---

## Features

- Researcher authentication and secure login
- Create and manage cognitive experiments
- Define trials and experimental tasks
- Collect participant responses
- Cloud database integration with MongoDB Atlas
- Browser-based experiment execution
- Dashboard for experiment management
